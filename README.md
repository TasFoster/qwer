# 🔥 Гриль-дом «от Элтаджа»

*Интернет-магазин шаурмы и шашлыка в эстетике угля и пламени — тёмный графит, огненные акценты, живая типографика (Unbounded · Golos Text · Caveat).*

![Pillow](https://img.shields.io/badge/Pillow-10%2B-376E99)

## ✨ Возможности

- Каталог с категориями (шаурма, шашлык, напитки, соусы) и фильтрацией по категории
- Карточки товаров с тегами «Хит» / «Остро», весом/объёмом и ценой
- Страница товара с выбором количества и блоком «С этим берут» (рекомендации из той же категории)
- Корзина на сессиях — покупки без регистрации
- Оформление заказа: доставка/самовывоз, оплата картой или наличными при получении, комментарий
- Управление меню и заказами через админку Django
- Адаптивная вёрстка, анимации появления, эмодзи-плейсхолдеры для блюд без фото

## 🛠 Стек

- **Backend:** Python · Django 6.0
- **База данных:** SQLite
- **Изображения:** Pillow (загрузка фото блюд)
- **Frontend:** серверные шаблоны Django, чистый CSS и JS, без сборщиков
- **Корзина:** хранится в сессии (ключ `CART_SESSION_ID = 'cart'`)
- **Локаль:** `ru-ru`, таймзона `Europe/Moscow`

## 🚀 Установка и запуск

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python manage.py migrate
python manage.py seed          # демо-меню: 4 категории, 16 блюд
python manage.py runserver
```

- Сайт: http://127.0.0.1:8000/
- Админка: http://127.0.0.1:8000/admin/

## 🧩 Структура

```
eltaj/                            настройки проекта (settings, urls, wsgi)
shop/
  models.py                       Category, Product, Order, OrderItem
  cart.py                         корзина в сессии
  views.py                        меню, товар, корзина, оформление
  forms.py                        форма заказа с валидацией адреса
  admin.py                        управление меню и заказами
  context_processors.py           корзина в контексте всех шаблонов
  templates/shop/                 base, menu, product_detail, cart, checkout, order_done
  management/commands/seed.py     наполнение демо-данными
static/css/style.css              вся стилизация
static/js/main.js                 степпер количества, авто-скрытие сообщений
media/products/                   фото блюд, загруженные через админку
```

### Модель данных

- **Category** — название, slug, порядок вывода.
- **Product** — категория, описание, вес/объём, цена, фото, флаги «Хит» / «Острое» / «В наличии».
- **Order** — имя, телефон, адрес, способ получения (доставка/самовывоз), оплата (наличные/карта), комментарий, статус оплаты.
- **OrderItem** — позиция заказа с зафиксированной ценой и количеством.

## 📷 Добавление фото блюд

Загрузите изображение в поле «Фото» товара в админке — оно автоматически заменит
эмодзи-плейсхолдер на карточке и странице товара. Файлы сохраняются в `media/products/`.
