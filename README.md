# 🔥 Гриль-дом «от Элтаджа»

Интернет-магазин по продаже шаурмы и шашлыка на Django 6.0.
Дизайн в эстетике «угля и пламени»: тёмный графит, огненные акценты,
живая типографика (Unbounded · Golos Text · Caveat).

## Возможности

- Каталог с категориями (шаурма, шашлык, напитки, соусы) и фильтрацией
- Карточки товаров с тегами «Хит» / «Остро», весом и ценой
- Страница товара с выбором количества и блоком «С этим берут»
- Корзина на сессиях (без регистрации)
- Оформление заказа: доставка/самовывоз, оплата картой/наличными
- Заказы и каталог управляются через админку Django
- Адаптивная вёрстка, анимации появления, эмодзи-плейсхолдеры без фото

## Запуск

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py seed          # демо-меню: 4 категории, 16 блюд
python manage.py runserver
```

Сайт: http://127.0.0.1:8000/ · Админка: http://127.0.0.1:8000/admin/

## Структура

```
eltaj/            настройки проекта (settings, urls)
shop/
  models.py       Category, Product, Order, OrderItem
  cart.py         корзина в сессии
  views.py        меню, товар, корзина, оформление
  forms.py        форма заказа с валидацией адреса
  admin.py        управление меню и заказами
  templates/shop/ base, menu, product_detail, cart, checkout, order_done
  management/commands/seed.py   наполнение демо-данными
static/css/style.css            вся стилизация
static/js/main.js               степпер количества, авто-скрытие сообщений
```

## Добавление фото блюд

Загрузите изображение в поле «Фото» товара в админке — оно автоматически
заменит эмодзи-плейсхолдер на карточке и странице товара.
Файлы сохраняются в `media/products/`.
